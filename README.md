## 准备

1. 安装 node 和 git 环境。

2. 对 ES2015, Vue 2.x, Vuex, Vue Router, Vue CLI 和 axios 有基本的了解。

## 开始

```
# 克隆项目
git clone https://github.com/zhangtuo1999/shangpinhui.git

# 进入项目目录
cd shangpinhui

# 安装依赖
npm install

# 运行项目
npm run serve
```

在浏览器中打开：http://localhost:8080

## 构建

```
# 构建项目
npm run build
```
